JS Flows and Loops!

Javascript much like other programming languages is read from top to bottom. However control flow breaks that top to bottom reading method by implementing key phrases to take control of the flow with which the code is read.Some examples of these phrases are if and else statements . Loops are well, loops.

But what kind of loops are there? There are For loops,While loops, Do loops and If-else loops and that's just to name a few. Now as for an example of loops in everyday life just think of walking.Right foot forward when right foot reaches optimal length right foot stops and left foot starts.Repeat until desired results.

What is The DOM?!

It’s basically just the entire site from the start to the end, as well as all else in between.Thats a really poor explanation of what the DOM is, so instead think of a web site as something with layers. The very first being the window that you use to open the program that you code with.The next being the document that everything is stored on.And the chain of layers continue.
